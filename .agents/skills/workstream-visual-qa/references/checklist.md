# Visual QA Checklist

Before completion:

- Run typecheck, lint, unit tests, and Playwright checks.
- Inspect desktop, tablet, and mobile renderings.
- Confirm there is no horizontal scroll caused by layout overflow.
- Confirm dense tables and queues remain readable.
- Confirm status badges and action controls do not resize rows unpredictably.
- Confirm modal/dialog focus trapping and escape behavior.
- Confirm icon-only buttons have accessible names and visible focus.
- Confirm form errors are tied to fields.
- Confirm empty states are useful and not decorative filler.
- Confirm all visible language uses Workstream product nouns.
- Confirm the UI does not rely on purple gradients, glow blobs, glass cards, or marketing layout patterns.
